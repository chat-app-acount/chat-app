通知を確実に受け取るための設定

以下の3つを設定すると、ブラウザを閉じていてもPCを再起動しても通知が届くようになります。

1
通知を許可する
ブラウザ上部に出る「通知を許可しますか？」で「許可」をクリック。
許可するとメッセージが届いたとき通知が表示されます。
2
バックグラウンド実行をオンにする
これをオンにするとタブやブラウザを閉じていても通知が届くようになります。
Chrome の場合
右上 ⋮ → 設定 → システム →「Googleがバックグラウンドアプリを実行し続けることを許可する」をオン
Edge の場合
右上 ⋯ → 設定 → システムとパフォーマンス →「Edgeが閉じられた後もバックグラウンドのアプリを実行し続ける」をオン
3
PC起動時にブラウザを自動起動する
PCを再起動しても自動でブラウザが起動して通知が届くようになります。
Chrome の場合
右上 ⋮ → 設定 → 起動時 →「特定のページを開く」→「新しいページを追加」に
https://chat-app-acount.github.io/chat-app/ を追加
Edge の場合
右上 ⋯ → 設定 → 起動時 →「これらのページを開く」→「新しいページを追加」に
https://chat-app-acount.github.io/chat-app/ を追加
3つ全て設定完了でブラウザを閉じていてもPCを再起動しても通知が届きます
